# PCLP - Partial

In test.c este rezolvarea problemei in mod "standart" , adica nu suntem limitati la a folosi doar o singura solutie.

main.c este ceea ce ni sa cerut la partial ; suntem limitati la a folosi o singura functie.

Urmeaza sa termin si problema main.c


## Utilizare

Am creat un script pentru a putea usura putin executia programului .

Sriptul se poate utiliza astfel : 

```bash
./script.sh test # pentru a rula test.c
```
```bash
./script.sh main # pentru a rula main.c
```
